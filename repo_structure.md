# شرح هيكل مشروع الأداة (pdfhl-trans) لتجهيزه للرفع على GitHub

يهدف هذا الملف إلى توضيح هيكل الملفات والمجلدات التي يتكون منها المشروع، وما هو دور كل ملف، ليكون دليلاً مرجعياً لك.

```text
/pdfhl-trans (المجلد الرئيسي للمشروع)
│
├── .gitignore          # ملف أداة Git: يخبر المستودع بتجاهل رفع الملفات غير الأساسية (مثل الكاش أو البيئات الوهمية)
├── LICENSE             # رخصة المشروع (MIT): تسمح للآخرين باستخدام الأداة بالكود وتحدد حقوقك الكاملة كمالك
├── README.md           # الدليل التعريفي الشامل: واجهة مشروعك على جيتهاب التي تعرض الميزات وطريقة التثبيت
├── pyproject.toml      # ملف إعدادات "بناء الحزمة": يحتوي على اسم الأداة، رقم الإصدار، والمكتبات المطلوبة لتشغيلها
│
└── pdfhl_trans/        # 📁 المجلد الرئيسي لكود الأداة (الحزمة البرمجية)
    │
    ├── main.py                     # 📄 نقطة الانطلاق الأساسية (Entry point) التي يتم تشغيلها عند كتابة اسم الأداة
    ├── __init__.py                 # 📄 ملف يخبر لغة بايثون أن هذا المجلد هو حزمة برمجية (Package)
    │
    ├── cache/                      # 📁 مجلد التخزين المؤقت
    │   ├── __init__.py
    │   └── translation_cache.py    # 📄 يدير قاعدة بيانات SQLite لحفظ الترجمات السابقة وتجنب استهلاك رصيد API للترجمات المكررة
    │
    ├── cli/                        # 📁 مجلد واجهات المستخدم في موجه الأوامر (Terminal)
    │   ├── __init__.py
    │   └── interactive_cli.py      # 📄 يحتوي على القوائم التفاعلية والأسئلة لتقديم تجربة استخدام ممتازة للمستخدم
    │
    ├── config/                     # 📁 مجلد الإعدادات
    │   ├── __init__.py
    │   └── settings.py             # 📄 يدير ملف الإعدادات السري الخاص بالمستخدم (إدارة مفتاح API، واللغة الافتراضية)
    │
    ├── core/                       # 📁 قلب الأداة للعمليات الأساسية على الـ PDF
    │   ├── __init__.py
    │   ├── annotation_writer.py    # 📄 مسؤول عن كتابة الترجمات وإضافتها كملاحظات (Notes/Pop-ups) داخل ملف الـ PDF
    │   ├── highlight_extractor.py  # 📄 مسؤول عن قراءة ملف الـ PDF واستخراج النصوص المظللة مع أخذ السياق المحيط بها بعين الاعتبار
    │   └── pdf_processor.py        # 📄 المنسق (Orchestrator) الذي يدير عملية القراءة والترجمة والكتابة بشكل متوازي (Multi-threading) السريع
    │
    ├── translation/                # 📁 مجلد الترجمة والذكاء الاصطناعي
    │   ├── __init__.py
    │   ├── base_translator.py      # 📄 قالب أساسي لمترجم (يسمح مستقبلاً بإضافة ChatGPT أو Claude بسهولة كبدائل)
    │   └── gemini_translator.py    # 📄 الكود الخاص بالاتصال المباشر بـ Google Gemini API واصدار طلبات الترجمة
    │
    └── utils/                      # 📁 مجلد الأدوات المساعدة
        ├── __init__.py
        ├── logger.py               # 📄 مسؤول عن تنسيق طباعة النصوص والخلفيات الملونة والرسائل للمستخدم في التيرمنال
        └── text_utils.py           # 📄 خوارزميات ذكية وضرورية لإصلاح النصوص العربية (RTL) لضمان عدم ظهور الحروف مقلوبة أو معكوسة في المخرجات
```

### 🚫 ملفات ومجلدات موجودة محلياً بجهازك ولن يتم رفعها (مكتوبة في `.gitignore`):
- `pdfhl_trans.egg-info/`: مجلد يتم إنشاؤه تلقائياً عند التثبيت ويفيد لتعريف اسم الحزمة وإصدارها.
- `.venv/`: مجلد البيئة الوهمية الذي يحتوي على كامل مكتبات بايثون المحملة، وحجمه كبير جداً.
- `.ruff_cache/` و `__pycache__/`: مجلدات تخزين مؤقت تصنعها بايثون وأداة ruff لتسريع الفحص والتشغيل ولن نحتاجها على جيتهاب.
